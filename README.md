# 🎰 台彩全能大數據分析引擎 (Taiwan Lottery AI)

這是一個基於 Python 與 Streamlit 開發的即時彩券分析與預測系統。
我們將冰冷的「歷史大數據」與有趣的「數位玄學」完美結合，並採用 **無伺服器架構 (Serverless)**，將爬蟲數據無縫同步至 Google 試算表，打造出最獨特、最全面的選號輔助工具！

## 🌟 核心亮點 (Core Highlights)

* **☁️ 雲端資料庫 (Google Sheets Integration)**：捨棄傳統 CSV，將歷史開獎數據與你的「AI 預測紀錄」自動寫入 Google 試算表，資料永久保存不遺失。
* **⏳ 時光倒流回測 (Backtesting)**：利用最新一期的開獎前數據，模擬 AI 預測並直接與實際開獎「對答案」，準確率一目了然！
* **🎯 拖牌命中率分析**：不僅給出拖牌號碼，更直接攤開歷史數據，告訴你「上一期開出 X，下一期開出 Y 的機率是多少 %」。

---

## 🚀 雙引擎分析模式

本系統提供兩種截然不同的選號流派，滿足所有的彩迷：

### 📊 第一引擎：理性數據分析 (科學算牌)
* **自動爬蟲更新**：一鍵啟動 Headless 瀏覽器抓取最新期數，自動補齊缺失資料。
* **四大 AI 預測策略**：
  1. 🔥 **全熱門號**：擷取近期最高頻率出現的號碼。
  2. ❄️ **全冷門號**：反向操作，鎖定極少出現的遺漏號碼。
  3. 🌗 **冷熱各半**：平衡風險的混和選號。
  4. 🧩 **拖牌精選**：基於歷史軌跡的關聯性矩陣，計算最容易被「連帶抽出」的號碼。
* **一鍵紀錄雲端**：算好的號碼捨不得忘記？點擊按鈕直接存入你的專屬 Google 預測日記。

### 🔮 第二引擎：感性玄學偏方 (直覺靈感)
* 🔯 **本日星座幸運靈數**：以日期與星座產生宇宙共振種子。
* 🙏 **數位擲筊求明牌**：模擬傳統求籤，連續擲出聖筊才給號。
* 🔢 **姓名/靈感測字**：將你的靈感文字透過 SHA-256 雜湊轉換為幸運數字。
* 📸 **靈感顯影 (AI 圖像選號)**：上傳一張有感覺的照片，解析像素能量轉化為號碼。
* 🔮 **易經六十四卦靈籤**：六次起卦，陰陽爻動態生成對應靈數。
* 📈 **股市代碼共振**：輸入股票代碼 (如 2330)，結合大盤財氣演算。
* ⛩️ **日本神社御神籤**：線上搖籤筒，抽出大吉運勢與明牌。
* 📦 **萬物條碼解碼器**：掃描身邊商品的 SKU 貨號或條碼，解開隱藏財富密碼。

## 🛠️ 技術架構與套件

* **前端介面**：Streamlit
* **資料處理**：Pandas, Numpy
* **自動化爬蟲**：Selenium, Chromium (Headless)
* **雲端資料庫**：Google Sheets API (`gspread`, `google-auth`)
* **影像解析**：Pillow (PIL)
